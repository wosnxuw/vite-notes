# VPN 协议一览

### 协议一览

（当前内容截止到 2025 年年末）

SS ShadowSocks 2013 年，第一次提出了分流的概念，2015 年死死喵。按照现在的说法，这个协议会被秒封。你买的机场如果还是这个协议，说明是先中转到国内服务器，再代为路由出去

VMess ：伪装。需要购买域名，配置 CDN 来隐藏 IP。本身加密，实际上是 TLS in TLS，两次加密，因而数据特征明显

Trojan：伪装为完整的 https 网站

VLESS：传输协议和加密层解耦，其 UUID 仅作身份认证及流量管理，协议本身不加密，必须要求传输层安全，否则相当于裸奔

XTLS：V2Fly 社区的一次分裂

Reality：直接借用别人的官网

Hysteria：利用 UDP 的一种协议
