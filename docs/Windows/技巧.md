### 传输大型数据集

传输数据集的时候，可以通过 Windows 网络文件夹共享

不需要有路由器，只需要两个电脑用网线连起来就行（当然有路由器更好）

两者之间会使用 APIPA（Automatic Private IP Addressing）来自动分配 IP 地址，没有网管，但是在同一个网段下（比如我这里分配到 169.254.111.5 和 169.254.174.197 两个保留地址）

对方计算机，选一个文件夹，属性为共享，然后添加 Everyone 即可

然后通过”运行“，输入 `\\169.254.111.5`  访问对方资源，这里要求输入对方计算机的”凭证“

如果对方登录了 Microsoft 账号，则输入邮箱和邮箱密码。此时输入计算机的用户名、登录密码都是不行的。（因为默认你的网络和共享中心里，是设置了”有密码保护“）

↑ 这里的微软账号登录，不需要计算机连网

如果没登 Microsoft，抱歉不知道。

（输入 cmd 中`whoami`返回的结果试一试？，类似`desktop-b6mln8c\name`或者`name`？）

不想继续共享了，选高级共享，上面有一个复选框，取消勾选。

### 休眠和睡眠

windows 电脑有休眠和睡眠

从动物的角度来看，休眠会度过一个冬天

而睡眠一般只度过一个晚上

所以

休眠：电脑彻底关机，把应用写入硬盘，下次开机还在。下次开机约等于重启了。图标是一个钟表

睡眠：内存还没断电，从中恢复也比较的快。图标是一个月亮，表示一个晚上

在控制面版里，可以定义电源按钮，这样你才可以显示休眠的按钮（正常只有睡眠）

第一个可控制的地方位于 设置--电源和睡眠

第二个可控制的地方位于 电源计划--是计算机进入睡眠状态

每个挡位可以单独的设置自己的计划，这俩个位置操纵的是同一个选项，改一个，你会发现另外一个也被更新了

我这里，网线可以唤醒电脑，导致电脑刚睡眠就立刻被唤醒

https://answers.microsoft.com/en-us/windows/forum/all/windows-wont-sleep/7fa65b05-a195-44f0-bcc6-fa0b946b90b5

这里给出的建议非常有用

```cmd
powercfg /devicequery wake_armed
```
