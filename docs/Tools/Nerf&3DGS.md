高斯泼溅很火，这里学习一下

### NeRF

Neural Radiance Fields 是神经辐射场的缩写，这是 2020 年提出的一个技术

它本身是一套方法，并不是像 Transformer 那样，是一个固定的架构，它里面只有一个 MLP，它是一整套流程（包括如何渲染这个 MLP）

它是，学习 2D 照片，建模一个 3D 场景

它是用一个多层感知机 MLP 来学习 3D 场景

输入是 5D：包括位置 xyz 和相机观察方向

输出是：RGB 和 alpha

沿着视线累加所有的点，得到最终显示器上 2D 平面的颜色

也就是说，它把整个 3D 空间，隐式的表达在了这个 MLP 里

问题：训练耗时极久

### 3DGS

3D Gaussian Splatting

2023 年发布的 3D 场景表示与实时渲染方法

它用大量带颜色、带形状的 3D 高斯椭球体去拟合一个场景

核心：场景直接显式的表达在这群高斯椭球里，直接梯度下降这些点的参数

泼溅：描述的就是 3D 点，投影到 2D 屏幕的过程

训练更快

高斯泼溅，是稀疏表示，因为一开始会简单拟合，空气的位置都没有椭球，只有边缘有椭球，训练中，还会剔除透明的椭球

而 NeRF 则是稠密，空间中任何一个点，都可以查询其透明度，空气也被表示，只是透明了
