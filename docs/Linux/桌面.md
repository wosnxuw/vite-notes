### 桌面环境 DE

**X Window System 或 Wayland**：这是图形显示的基础。X Window System（通常称为 X11）是传统的显示服务器，而 Wayland 是现代替代品。它们负责管理图形显示、输入设备等。

**桌面环境（Desktop Environment）**：提供完整的用户界面，包括窗口管理器、面板、菜单、文件管理器、终端等一系列应用程序和工具。常见的桌面环境有 GNOME、KDE、XFCE、LXDE 等。

**GTK**： 主要用于 GNOME、XFCE
**Qt**： 主要用于 KDE Plasma

GNOME，现代，简洁且高效

KDE，可自定义程度高

XFCE，轻量级

Ubuntu Gnome

XUbuntu XFCE

KUbuntu KDE

Debian 默认是 Gnome

CentOS Gnome

OpenSUSE KDE Plasma

### 防火墙

iptables Linux 内核里的防火墙，但是用起来麻烦。在老的发行版上，就是内核；在新的发行版上，内核是 nftables，iptables 则是一个假命令，兼容层，转给其它命令处理

Ubuntu/Debian (UFW)

RHEL/CentOS/Fedora (FirewallD)，firewalld 的 d 是 deamon，就是后台服务，管理的工具则是 firewall-cmd

反正先用看服务是否已经跑起来，然后 ss 看是否监听端口，最后用 ufw 打开端口

`ss -tlnp | grep :22`

`firewall-cmd --list-ports`

`firewall-cmd --add-service=ssh --permanent`

`firewall-cmd --reload`

### systemd

现代 Linux 系统的“第一个进程”（PID 1）

功能：系统初始化、服务管理 (nginx)、挂载文件系统、网络配置

systemctl 是 systemd 的控制工具

现在基本上我这种小白能说的上来的 Linux 都有 systemd。systemd 被人诟病就是说它比较臃肿，管理的事情太多

`systemctl enable xxx` 多数时候相当于设置这些的单元为开机自启动（但不会立即启动）
`systemctl disable xxx`
`systemctl is-enabled xxx` 返回 0 是有开机自启（但不一定当前就在工作）
`systemctl status xxx` 现在是否在跑
`systemctl start xxx` 跑起
