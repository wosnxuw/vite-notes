### 桌面环境 DE

**桌面环境（Desktop Environment）**：提供完整的用户界面，包括窗口管理器、面板、菜单、文件管理器、终端等一系列应用程序和工具。常见的桌面环境有 GNOME、KDE、XFCE、LXDE 等。

桌面环境 = 窗口管理器 + 面板 + 启动器 + 设置 + 文件管理器 + 各种后台服务

**GTK**： 主要用于 GNOME、XFCE

**Qt**： 主要用于 KDE Plasma

GNOME，现代，简洁且高效

KDE，可自定义程度高

XFCE，轻量级

Debian       Gnome（默认）

CentOS       Gnome

Ubuntu       Gnome

XUbuntu      XFCE

KUbuntu      KDE

OpenSUSE     KDE Plasma

### 窗口管理器 WM

主要负责窗口的行为

浮动式窗口管理器：

Mutter      GNOME 默认

KWin        KDE Plasma 默认

Xfwm        Xfce 默认

平铺式窗口管理器：Hyprland 等

### 显示服务器 显示协议

以前：X11 / Xorg

现代：Wayland

XWayland 是 Wayland 对老应用等兼容

它的作用：管理屏幕、接收键盘鼠标输入、给应用提供画图接口、维护窗口对象

X11 时代的流程：

应用程序
  │
  ▼
X Server / Xorg
  │
  ▼
窗口管理器 WM
  │
  ▼
合成器 Compositor（可选，负责透明效果、阴影等）
  │
  ▼
显卡驱动
  ▼
屏幕

Wayland 时期流程：

应用程序
  │
  ▼
Wayland Compositor
  │
  ▼
内核 DRM/KMS + 显卡驱动
  ▼
屏幕

Wayland 合成器通常同时就是窗口管理器和显示服务器，传统上 X11 里那些分开的角色被合并了

### Display Manager 登陆界面

eg：GDM, SDDM, LightDM

作用：显示登录界面、选择用户、输入密码，登录后启动桌面环境或窗口管理器

GDM  常用于 GNOME
SDDM 常用于 KDE
LightDM 常用于 Xfce

systemd 启动 DM，DM 启动 DE/WM
